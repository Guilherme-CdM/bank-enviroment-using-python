# Introdução

    Projeto Full-Stack para simular ações bancárias utilizando banco de dados local combinando Python e Node.js

## Versão
    | BETA | Página estática com SEO |

## Requisitos
    | Componente | Versão recomendada |
    |---|---|
    | Node.js | LTS (>=18.x) |
    | NPM | >=8.x |
    | Python | >=3.9 |
    | HTML | HTML5 |
    | CSS | CSS3 |
    | Visual Studio Code | Última versão estável |
    | Git | >=2.30 |

## Objetivo
    Criar e desenvolver um sistema bancário funcional simples utilizando linguagem POO (Programação Orientada a Objetos) no Back-End e, boas práticas junto com SEO, no Front-End.
